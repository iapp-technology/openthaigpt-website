# Computing Resources

โครงการ OpenThaiGPT ได้รับการสนับสนุนทรัพยากรประมวลผล (GPU Farm) จาก [องค์กรที่ร่วมสนับสนุนโครงการ (Sponsors)](../../sponsors.md) เพื่อใช้ในการพัฒนา Model OpenThaiGPT โดยการเข้าถึงทรัพยากรเหล่านี้เปิดให้เฉพาะอาสาสมัครที่ลงทะเบียนกับโครงการเท่านั้น (Volunteer-Only)

หากสนใจเข้าถึงทรัพยากรประมวลผลเพื่อร่วมพัฒนา OpenThaiGPT สามารถสมัครเป็นอาสาสมัครได้ที่หน้า [ร่วมกับเรา (Join Us)](../../join-us.md)
