# Open Resources

| Related Paper / Knowledge          | รวบรวม Paper หรือองค์ความรู้ทางด้าน LLM และ Transformers                                                          | Public         | [Related Paper / Knowledge](related-paper-knowledge.md)                                                                                                           |
