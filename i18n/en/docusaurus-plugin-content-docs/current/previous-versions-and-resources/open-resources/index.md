We have gathered all the resources used to build OpenThaiGPT. Our Open Resources are made available according to the following access rights.

| Released Models                    | Models developed by the OpenThaiGPT community                                                                     | Public         | [Released Models (0.1.0-beta, 16/05/23)](../previous-events/released-models-version-less-than-0.1.0-beta-greater-than-16-05-23.md)  |

| Related Paper / Knowledge          | A collection of papers and knowledge on LLMs and Transformers                                                     | Public         | [Related Paper / Knowledge](related-paper-knowledge.md)                                                                                                           |
| Free Working Datasets              | A collection of other open datasets that can be used to train OpenThaiGPT                                         | Public         | [Free Working Datasets](free-working-datasets.md)                                                                                                               |
