# Computing Resources

The OpenThaiGPT project receives computing resources (GPU Farm) from [organizations supporting the project (Sponsors)](../../sponsors.md) for use in developing the OpenThaiGPT model. Access to these resources is available only to volunteers registered with the project (Volunteer-Only).

If you are interested in accessing computing resources to help develop OpenThaiGPT, you can apply to become a volunteer on the [Join Us](../../join-us.md) page.
