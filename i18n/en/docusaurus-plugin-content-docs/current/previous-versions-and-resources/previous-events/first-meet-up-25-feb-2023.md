## Agenda

| Time        | Agenda                                                                                                                                                                     |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 13:00-13:15 | ดร.เทพชัย + ดร. กอบกฤตย์ explain the background and origins of OpenThaiGPT                                                                                                 |
| 13:15-14:00 | ดร. สุเมธ explains how ChatGPT works and how to build a ChatGPT                                                                                                            |
| 14:00-14:15 | ดร. กอบกฤตย์ explains how OpenThaiGPT version 0.0.1 was built, shares a Colab, and walks through the process                                                               |
| 14:15-15:00 | ดร. กอบกฤตย์ leads a discussion on how to collect Thai InstructGPT data, building a website to help collect data collaboratively, and guidelines for rapidly collecting the Thai InstructGPT Dataset |
| 15:00-16:00 | ดร. สุเมธ + ดร.ปรัชญา lead a discussion on the pretraining approach — which model to choose, which dataset to train on, and which GPU server to use for training            |
| 16:00-16:30 | ดร. กอบกฤตย์ explains how to do Reinforcement Learning with Human Feedback, with an example Colab, and leads a discussion on how to develop a reward model for PPO          |
| 16:30-17:00 | Volunteer team registration, appointing team leads, setting the timeline and weekly meeting schedule, and officially beginning OpenThaiGPT development                      |
