# 协作规则（Rules）

为营造良好的协作环境，让社会各界都能高效地参与 OpenThaiGPT 的开发，并共同确立明确的目标，我们制定了以下三条协作规则。

1. 项目产出的所有成果必须遵循以下授权协议
   1. Source Code / Weight / Model = Apache 2.0
   2. Dataset = CC BY-SA
2. 资源访问（Open Resource）依照 [Open Resources](previous-versions-and-resources/open-resources/index.md) 页面所列信息执行，其中部分资源仅向志愿者群体开放，申请流程如下
   1. 注册成为志愿者团队的一员
   2. 按照所属志愿者团队负责人的要求，以任意方式开始为项目做出贡献，例如
      1. 数据标注网站团队：
         1. 为数据标注网站的开发贡献至少 1 个 commit
      2. InstructDataset 团队：
         1. 协助标注至少 10 组对话对的 InstructDataset
      3. RLHF 团队：
         1. 协助对模型生成结果进行排序，以构建奖励模型，至少完成 10 组对话对
      4. 预训练团队：
         1. 协助清洗预训练数据，至少 10 篇文章
         2. 参与实验以寻找合适的 LM 结构，至少 1 种方案
      5. OpenThaiGPT Library 开发团队：
         1. 协助开发 PIP OpenThaiGPT Library，至少 1 个 commit
      6. 其他形式的适当贡献
   3. 由志愿者团队负责人将名单提交给协调团队，以便更新网站上的名单并开放后续的数据访问权限
3. 如需以 OpenThaiGPT 项目名义申请经费或谋求任何其他利益，必须取得 OpenThaiGPT 协调团队出具的推荐函，该函件仅由泰国人工智能创业者协会（AIEAT）与泰国人工智能协会（AIAT）共同签发。
