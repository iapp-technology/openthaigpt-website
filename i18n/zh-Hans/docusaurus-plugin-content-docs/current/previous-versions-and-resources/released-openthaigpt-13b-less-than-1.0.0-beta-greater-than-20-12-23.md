# 🔥 发布 OpenThaiGPT 13b <1.0.0-beta>（20/12/23）

🇹🇭 OpenThaiGPT 13b 1.0.0-beta（2023 年 12 月 20 日）

🇹🇭 OpenThaiGPT 13b 1.0.0-beta 版本是一款 130 亿参数的泰语 LLaMA v2 Chat 模型，在经过清洗的 150 亿 token 泰语文本数据集上进行了继续预训练，并针对泰语翻译指令进行了微调，同时将 10,000 多个最常用的泰语词汇扩充进大语言模型的词表，以实现极速推理。

## 更新日志

### 🇹🇭 OpenThaiGPT 13b 1.0.0-beta 版本（Llama v2 + 150 亿泰语 token 预训练 + 10,000 泰语词汇扩充）

**发布日期：2023 年 12 月 20 日**
