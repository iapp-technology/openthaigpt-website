# 开放资源（Open Resources）

我们汇集了构建 OpenThaiGPT 所使用的全部资源，这些开放资源按照以下权限对外开放。

| 资源                               | 说明                                                                                                              | 开放对象       | 链接                                                                                                                                                                         |
| ---------------------------------- | ----------------------------------------------------------------------------------------------------------------- | -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Released Models                    | 由 OpenThaiGPT 社区开发的模型                                                                                     | 公开           | [Released Models (0.1.0-beta, 16/05/23)](../previous-events/released-models-version-less-than-0.1.0-beta-greater-than-16-05-23.md)  |
| Released Datasets                  | 由 OpenThaiGPT 社区开发的数据集                                                                                   | 公开           | [Released Datasets (14/04/23)](../released-datasets-14-04-23.md)                                                                                                  |
| Released Code & Colab              | 用于开发 OpenThaiGPT 模型的代码或 Colab                                                                           | 公开           | [Released Code / Colabs](../released-code-colabs/index.md)                                                                                                                   |
| Related Paper / Knowledge          | 汇集大语言模型与 Transformers 相关的论文与知识资料                                                                | 公开           | [Related Paper / Knowledge](related-paper-knowledge.md)                                                                                                           |
| Free Working Datasets              | 汇集其他可用于训练 OpenThaiGPT 的开放数据集                                                                       | 公开           | [Free Working Datasets](free-working-datasets.md)                                                                                                               |

| Computing Resources                | 汇总由赞助方提供的 GPU Farm 接入方式，用于开发 OpenThaiGPT 模型                                                   | 仅限志愿者     | [Computing Resources](computing-resources.md)                                                                                                                   |
