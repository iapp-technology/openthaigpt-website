# 计算资源

OpenThaiGPT 项目获得了[项目支持机构（Sponsors）](../../sponsors.md)提供的计算资源（GPU Farm），用于开发 OpenThaiGPT 模型。这些资源仅向已在项目中注册的志愿者开放（Volunteer-Only）。

如果您有意获取计算资源并参与 OpenThaiGPT 的开发，可以在[加入我们（Join Us）](../../join-us.md)页面申请成为志愿者。
