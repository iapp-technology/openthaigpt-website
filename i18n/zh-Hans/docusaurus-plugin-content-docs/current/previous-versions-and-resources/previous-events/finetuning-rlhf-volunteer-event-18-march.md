# 微调 / RLHF 志愿者活动（3 月 18 日）

请允许我在此发布一则通知。大家好，我是 Sumeth 博士。

诚邀对微调 / RLHF 志愿工作感兴趣的各位志愿者，参加微调 / RLHF 志愿者团队的 First Meet 活动，时间为 2023 年 3 月 18 日星期六 13:00-14:00。

本次活动的主要目标，是将全体志愿者分成若干小组，并推选出各小组的代表，共同推进微调 / RLHF 志愿工作的开展。

## First Meet 活动议程

* 13:00-13:30 介绍微调 / RLHF，说明其重要性以及当前的工作计划
* 13:30-13:40 介绍各小组及其职责
* 13:40-13:50 问答环节
* 13:50-14:00 志愿者报名加入意向小组
