## 日程安排

| 时间        | 日程内容                                                                                                                                                                   |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 13:00-13:15 | ดร.เทพชัย 与 ดร. กอบกฤตย์ 介绍 OpenThaiGPT 的缘起与背景                                                                                                                    |
| 13:15-14:00 | ดร. สุเมธ 讲解 ChatGPT 的工作原理以及构建 ChatGPT 的方法                                                                                                                   |
| 14:00-14:15 | ดร. กอบกฤตย์ 讲解 OpenThaiGPT version 0.0.1 的构建方法，并分享 Colab 与具体操作步骤                                                                                        |
| 14:15-15:00 | ดร. กอบกฤตย์ 主持讨论：如何采集 Thai InstructGPT 数据、开发协作式数据标注网站，以及快速构建 Thai InstructGPT Dataset 的规范指南                                             |
| 15:00-16:00 | ดร. สุเมธ 与 ดร.ปรัชญา 主持讨论：模型预训练的方法，包括选用哪个模型、使用哪些数据集进行训练，以及使用哪里的 GPU 服务器进行训练                                              |
| 16:00-16:30 | ดร. กอบกฤตย์ 讲解 Reinforcement Learning with Human Feedback 的实现方法并提供示例 Colab，同时讨论如何开发用于 PPO 的奖励模型                                                |
| 16:30-17:00 | 志愿者团队登记、确定团队负责人、制定时间表与每周例会安排，正式启动 OpenThaiGPT 的开发工作                                                                                  |
